RESPOSTAS

A)O "placeholder" especifica uma dica curta que descreve o valor esperado de um campo de entrada (um valor de amostra ou uma breve descrição do formato esperado).

B)É usada para habilitar explicitamente a verificação estouro para conversões e operações aritméticas de tipo integral.

C)O "value" especifica um valor inicial para um campo de entrada.

D)O atributo ACTION é quem diz para onde os dados de um formulário deve ser enviado, devemos informar a maneira como queremos que esses dados sejam enviados, ou seja, se queremos que eles sejam enviados através de uma requisição do tipo GET ou POST.

E)"Label" representa uma legenda para um item em uma interface de usuário, colocando o elemento "for" ele pode ser chamado de controle etiquetado.

F)Indica que várias opções podem ser selecionadas na lista.

G)É usado para indicar que um determinando campo de formulário é obrigatório para o envio do mesmo.

H)"Action" Define o local (através de uma URL) ao qual serão enviados todos os dados recolhidos do formulário.

I)O atributo "method" define qual o método HTTP para enviar os dados (ele pode ser "GET" ou "POST").

J)"input type=”password”"

K)O valor do atributo deve ser um id de um elemento "form".
